# My Learning Journal
A comprehensive, open-source repository documenting my self-study journey through technical textbooks, coding exercises, and synthesized research projects. 

This repository serves as the source code for my interactive Jupyter Book documentation.

# Repository Structure
The repository is divided into two distinct learning streams:
1. Textbook Exercises & Solutions
   This section contains book-by-book breakdowns of core academic texts. Each textbook folder includes:Exercise Solutions: Documented code and mathematical derivations solving end-of-chapter problems.Additional Projects: Small-scale code experiments directly inspired by the text.Core Takeaways: Synthesized notes and insights gained from each chapter.
2. Advanced Research Projects:
   This section contains standalone, advanced engineering and data science projects. Each project is structured like an academic paper or technical report:Title: A clear statement of the engineering or scientific problem.Literature Review: A deep, synthesized review of existing papers, books, and methodologies related to the topic.Recreation of Results: Hands-on code implementations that reproduce baseline data, models, or algorithms from the reviewed literature.
